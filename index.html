    สิ่งจำเป็นสำหรับใช้ LegendaryClient For Garena

    • Window 7 ขึ้นไป
    • .Net Framework 4.5

    วิธีใช้งาน LegendaryClient

    รัน Legendary Garena Launcher.exe จากนั้นรันเกม LoL ผ่าน Garena Plus ปกติ ตัวเกมปกติจะไม่ขึ้นมาแต่LegendaryClient จะรันขึ้นมาแทน รอจนกว่ามันจะ Update เสร็จ ครั้งแรกจะนานหน่อย ถ้ามันถามหาไฟล์ lol.exe ให้เลือก ไปที่ lol.exe ใน Folder ของ LoLTH เมื่อ Update เสร็จแล้วจะเข่้าสู้หน้าLogin ให้เลือกที่ Server TH และ กด Login ได้เลย

    • เฉพาะครั้งแรก เมื่อถึงหน้า Loginแล้ว ให้ปิดมันแล้วรันใหม่ 1 รอบ ** ถ้าขึ้น username or password incorrect ตอน Login แปลว่า tokenหมดอายุให้ปิดเกมแล้วเปิดใหม่
