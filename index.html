    สิ่งจำเป็นสำหรับใช้ LegendaryClient For Garena

    • Window 7 ขึ้นไป
    • .Net Framework 4.5
    • League of Legends NA Client (กำลังทำให้ใช้งานกับ GarenaClient)
    • Process Explorer หรือ TokenDump

    วิธีใช้งาน LegendaryClient

    1. รัน LCStartUpSplash.exe
    2. ด้านบน ตรงคำว่า LegendaryClient Updater จะมี Menu Boxให้เลือก ให้เลือก Live แล้วรอมัน Download Update
    3. เมื่อ Update เสร็จจะมี หน้าต่างขึ้นมาเรียกหา lol.launcher.exe ให้เลือกไปที่ lol.launcher.exe ของ NA Client จากนั้นมันจะ Check Version ต่อเมื่อ Check เสร็จจะเข้าสู่หน้า Login
    4. ในหน้า login ให้เลือก Server เป็น Garena และ เลือก เป็น TH
    5. Username ใส่ Username ทีใช้ login Garena ,ช่อง UID ใช้ UID ของ Garena , ช่อง GarenaToken ใส่ Token ที่ได้จาก Process Explorer หรือ TokenDump

    วิธีใช้งาน Process Explorer เพื่อหา Token

    1. เปิดโปรแกรม Process Explorer จากนั้นเปิดเกม ตามปกติจาก GarenaPlus (หาโปรแกรมได้จากตาม Google)
    2. หา Process lol.exe ในโปรแกรม Process Explorer ตามภาพ 3.จากนั้น Double Click Process lol.exe จะมีหน้าต่างใหม่ขึ้นมาตามภาพ
    3. Copy Token ทั้งหมด จาก ช่อง Command Line ไป Paste ใส่ ช่อง GarenaToken หน้า Loginของ LegendaryClient (อย่าลืมลบ ช่องว่าง ก่อนตัวหนังสือตัวแรกออกด้วย ปกติตอน Copy มามันจะติดมา เพราะถ้า Token ผิดจะ Loginไม่ได้) จากนั้น ปิดตัวเกม ปกติไปแล้ว กด Login

    วิธีใช้ TokenDump

    1. Download https://db.tt/SjVvIpXa
    2. แตกไฟล์เอาไปไว้ที่ LoLTH เช่น D:\GarenaLoLTH\GameData\Apps\LoLTH ให้ Rename lol.exe ของเก่า หรือ Backup เก็บไว้ด้วย ตอนจะเล่นปกติผ่านClientปกติต้องเอามาทับกลับคืน
    3. หลังจากเอา lol.exe จากในzipไปไว้แล้ว รันเกมปกติ จาก Garena Plus แต่ตัวเกมจะไม่มีไรขึ้นมา ให้เข้าไปที่ Drive C จะมีไฟล์ชื่อ Garenatoken.txt ให้เปิดไฟล์ แล้ว Copy ทั้งหมดไปใส่ ช่อง GarenaToken ในหน้าLogin ของ LegendaryClient
